## 問題文

traP にはデジタル創作初心者向けに 0→1 講習会というものがあります。

長さ *N* の 0 と 1 からなる文字列 *A* が与えられます。文字列の *i* 番目 (1 ≤ *i* ≤ *N*) の要素を *A<sub>i</sub>* とします。この文字列に対する、次の操作を考えます。

- 1 ≤ *i* ≤ *N* を満たす整数 *i* を選ぶ。
- *A<sub>i</sub>* = 0 かつ *i* < *N* なら、*A<sub>i+1</sub>* を 0 にする。
- *A<sub>i</sub>* = 1 かつ *i* > 1 なら、*A<sub>i−1</sub>* を 1 にする。

文字列 *A* からこの操作を 0 回以上行うことで作ることができる文字列の個数を 998244353 で割ったあまりを出力してください。

## 制約

- 1 ≤ *N* ≤ 5·10<sup>5</sup>
- *A* は長さ *N* の `0`, `1` のみで構成される文字列

## ヒント（解放済み）

A が `000111` の場合に作ることができる文字列がどのようになるかを考えてみましょう。
