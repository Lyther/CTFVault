## 問題文

東京科学大学デジタル創作同好会 traP では、部員一人ひとりが traQ ID と呼ばれるユーザー ID を持っています。traQ ID は以下の要件を満たす必要があります。

- 1 文字以上 32 文字以下の文字列である。
- 英大文字・英小文字・アラビア数字・アンダースコア・ハイフン `A-Z a-z 0-9 _ -` 以外の文字を含まない。
- 先頭および末尾の文字がアンダースコア・ハイフン `_ -` ではない。

これから traP に入部するあなたは、与えられる文字列 *S* を traQ ID として登録しようとしています。上の要件を満たしているならば `200` を、そうでないならば `400` を出力してください。

## 制約

- 1 ≤ |*S*| ≤ 100
- 文字列 *S* は ASCII code で 33〜126 と表現される文字から構成される。特に、**半角スペースを含まない**。
