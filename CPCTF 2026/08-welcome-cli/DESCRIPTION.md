インスタンスを起動して、コマンドをお手元の shell に入力し、パスワードを入力すると、このような表示がされると思います。

```shell
ssh2@shell-ssh-jwefairt-6f75f5cdb6-hb425:~$
```

これは linux と呼ばれるシステムです！linux では、コマンドを使って操作をするのですが、この問題を解くには `cat` というコマンドが必要です。`cat (中身を表示したいファイルのパス)` で、ファイルの中身を表示できます。

なお、linux のコマンドについて詳細を知りたい場合は、`man (詳細を知りたいコマンドの名前)` とコマンドを打つと公式のマニュアルが表示されます。また、そもそも使えるコマンド一覧を知りたい場合は、`compgen -c` とコマンドを打つと、現状の環境で使えるコマンドの一覧が表示されます。
