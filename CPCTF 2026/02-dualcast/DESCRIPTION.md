配布ファイルは、Python というプログラミング言語のコードです。Crypto では Python を使うことがほとんどなので、これを機にインストールしてみましょう！WSL をお持ちの方は、初めから python が入っているかもしれません。ちなみに Webshell にも入っています。

さて、コードを読んでみると、2 回の型変換が行われていることがわかります。具体的には、文字列からバイト列に、そしてバイト列から整数に変換されています。よってこの問題を解くにはその逆変換を行えばいいです。

ちなみに、Crypto の問題では、この問題のように `bytes_to_long`, `long_to_bytes` 関数を使うことで、平文を整数に変換し、暗号化をすることが多いです。(他の変換方法をする場合も当然あります) また、`b"文字列"` と書くことでも、文字列を byte 列に変換することもできます。(こちらの方が短く簡潔に書けるので使われることも多いです)
