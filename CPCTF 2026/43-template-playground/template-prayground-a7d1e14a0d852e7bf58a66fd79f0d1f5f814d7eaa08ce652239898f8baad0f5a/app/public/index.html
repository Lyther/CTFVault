Helper Functions

クリックしてテンプレートで使えるヘルパーを選択。選択した関数名が use に追加されます。

✓
upper(s)
文字列を大文字に変換します
"hello" → "HELLO"
✓
lower(s)
文字列を小文字に変換します
"HELLO" → "hello"
✓
reverse(s)
文字列を逆順にします
"abc" → "cba"
✓
repeat(s, n)
文字列を n 回繰り返します
"ab" × 3 → "ababab"
