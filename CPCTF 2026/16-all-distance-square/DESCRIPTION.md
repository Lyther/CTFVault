## 問題文

正整数 *N* が与えられます。*N* 頂点 *M* 辺の重み付き連結無向グラフであって、以下の条件を満たすものを一つ構築してください。ただし、頂点には 1 から *N* までの、辺には 1 から *M* までの番号がそれぞれ付けられているものとします。

- 自己ループを含まない (多重辺はあっても良い)
- *M* ≤ 2*N* − 3
- 1 ≤ *i* < *j* ≤ *N* を満たす全ての整数の組 (*i*, *j*) に対して、頂点 *i* から *j* への単純パスであって、そのパスに含まれる辺の重みの総和が平方数であるものが存在する
- 全ての辺の重みは相異なる 1 以上 200 以下の整数である

さらに、あなたが構築したグラフにおける、頂点 *i* から *j* へ向かう単純パスであって、そのパスに含まれる辺の重みの総和が平方数であるものの一つを *P<sub>i,j</sub>* とします。また、*P<sub>i,j</sub>* が通る辺の番号を通る順に並べた数列を *Q<sub>i,j</sub>* とします。1 ≤ *i* < *j* ≤ *N* を満たす全ての整数の組 (*i*, *j*) に対して、*Q<sub>i,j</sub>* を出力してください。

なお、条件を満たすグラフは必ず構築可能であることが示せます。

## 制約

- 2 ≤ *N* ≤ 100
- 入力は全て整数

## ヒント（解放済み）

まずは、条件のうち三つ目を少し緩めて、

- 2 ≤ *j* ≤ *N* を満たす全ての整数 *j* に対して、頂点 1 から *j* への単純パスであって、そのパスに含まれる辺の重みの総和が平方数であるものが存在する

としたときの解法を考えてみましょう。それだけで選択肢はかなり絞られるはずです。
