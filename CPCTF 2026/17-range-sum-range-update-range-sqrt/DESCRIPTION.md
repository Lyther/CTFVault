## 問題文

長さ *N* の配列 *A* = (*A*<sub>0</sub>, *A*<sub>1</sub>, ..., *A*<sub>*N*−1</sub>) が与えられます。ここで、各 *A<sub>i</sub>* は 0 以上の整数です。
次の 3 種類のクエリを合計 *Q* 回処理してください。ここで、`isqrt(x)` は √*x* の整数部分を返す関数、つまり `isqrt(x) := ⌊√x⌋` です。

- クエリ 0 (`0 l r`): *A<sub>l</sub>* + *A<sub>l+1</sub>* + ... + *A<sub>r−1</sub>* を求める。
- クエリ 1 (`1 l r x`): *A<sub>l</sub>*, *A<sub>l+1</sub>*, ..., *A<sub>r−1</sub>* に *x* を代入する。
- クエリ 2 (`2 l r`): *A<sub>l</sub>*, ..., *A<sub>r−1</sub>* の全てに `isqrt` を作用させる。すなわち、*l* ≤ *i* < *r* を満たす全ての *i* に対して、*A<sub>i</sub>* を ⌊√*A<sub>i</sub>*⌋ に置き換える。

## 制約

- 1 ≤ *N* ≤ 10<sup>5</sup>
- 1 ≤ *Q* ≤ 10<sup>5</sup>
- 0 ≤ *A<sub>i</sub>* ≤ 10<sup>9</sup>
- 0 ≤ *l* ≤ *r* ≤ *N*
- 0 ≤ *x* ≤ 10<sup>9</sup>
- 入力は全て整数

## ヒント（解放済み）

このような区間に対する操作を行うデータ構造としては遅延セグメント木が有効です。しかし、「和を取ってから isqrt を取る場合」と「isqrt を取ってから和を取る場合」とでは値が変わってしまいます。どうすればよいでしょうか？
